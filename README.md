# RT_CV

# Система управления лампочками через жесты

Программный комплекс для распознавания жестов руки через веб-камеру с последующим визуальным отображением состояния пяти лампочек в веб-интерфейсе.

## Описание компонентов

| Файл | Назначение |
|------|-----------|
| `CV_main.py` | Захват видео с камеры, детекция кисти руки, подсчёт поднятых пальцев через MediaPipe |
| `agent.py` | Принятие решений на основе данных сенсора с использованием LLaMA 3.1, управление яркостью |
| `syte.html` | Веб-интерфейс с пятью индикаторами, отображающими текущее состояние |
| `utils.py` | Вспомогательные функции: логирование, проверка состояния Ollama |
| `hand_landmarker.task` | Предобученная модель MediaPipe для детекции рук |

## Системные требования

- Python 3.10 или выше
- Веб-камера
- 8 ГБ оперативной памяти (для работы LLaMA 3.1)
- ОС: Windows 10/11, macOS 12+, Ubuntu 20.04+

## Установка зависимостей

```bash
pip install opencv-python mediapipe websockets ollama
```

Установка Ollama согласно инструкции для вашей ОС.

Загрузка модели MediaPipe:

# Скачать hand_landmarker.task по ссылке:
# https://storage.googleapis.com/mediapipe-models/hand_landmarker/hand_landmarker/float16/latest/hand_landmarker.task
# Поместить файл в корневую директорию проекта

Загрузка языковой модели:
ollama pull llama3.1

Запуск системы
Требуется три отдельных терминальных сессии.

Терминал 1 — сервер Ollama:
ollama serve

Терминал 2 — сенсор:
python sensor.py
Откроется окно с видеопотоком. Для завершения нажмите q.

Терминал 3 — агент:
python agent.py
После запуска агента откройте site.html в браузере. Индикатор статуса в верхней части страницы должен сменить цвет на зелёный.

Поддерживаемые жесты

0 пальцев (кулак)	Все лампочки выключены
1–5 пальцев	Зажигается соответствующее количество лампочек
Кулак → ладонь	Переключение режима яркости (стандартный/тусклый)
Удержание позы	Состояние не меняется (используется кэширование)
